# Multilevel Feedback Queue Scheduling (C++)

This project implements a Multilevel Feedback Queue (MLFQ) CPU scheduling algorithm in C++.

## Features
- Three priority queues (Q1, Q2, Q3)
- Dynamic time quantum
- Priority promotion and demotion
- I/O-bound process priority boost
- Waiting Time and Turnaround Time calculation
- Queue movement history tracking

## How to Run
Compile using any C++ compiler:
g++ MLFQ_Scheduler.cpp -o mlfq
./mlfq

## Platform
- Windows
- Linux (Ubuntu)
